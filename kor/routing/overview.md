# Routing - 문서 찾기

## 이 폴더의 역할

상황에 따라 어떤 문서를 읽어야 하는지 안내합니다.

"지금 이 작업을 하려면 어떤 문서를 봐야 하지?"에 대한 답을 제공합니다.

## 어떤 문서가 들어갈까요?

예를 들어:
- 메인 라우팅 맵 (상황별 문서 안내)
- 문서 업데이트 가이드
- 문서 간 연결 규칙

## 핵심 아이디어

AI가 알아서 찾게 하는 것이 아니라,
명확하게 "이 상황에서는 이 문서를 읽어"라고 지정합니다.

## LLM 사용 지침

- **작업 시작 전**: 사용자 요청의 상황을 파악하고 이 폴더의 라우팅 맵에서 읽을 문서 목록 확인
- **순서 준수**: 라우팅 맵이 제시한 순서대로 문서를 읽어야 함
- **업데이트 시**: 문서 수정 후 동기화 규칙을 확인하여 연관 문서도 함께 갱신
