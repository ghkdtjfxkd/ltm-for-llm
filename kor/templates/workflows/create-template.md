# Create Template

> 새로운 템플릿을 작성하고 저장

## 입력

- `name`: 템플릿 이름
- `purpose`: 어떤 문서를 만들 때 쓰는지

## 실행

1. `templates-routing.md`에서 기존 템플릿 목록 확인
2. 새 템플릿 작성
   - `{}` 형식으로 채울 항목 표시
   - 최소한의 구조만 (과도한 항목 금지)
3. `templates/`에 저장
4. `templates-routing.md` 타입 테이블 업데이트
5. `overview.md` 목록 업데이트
6. `entry.md` 지원 작업 확인 (새 작업 유형이면 추가)

## 출력

- 생성된 템플릿 파일 경로
