# Meta - 시스템 가이드

## 이 폴더의 역할

시스템 전체를 이해하고 사용하기 위한 가이드 문서들이 있습니다.

새로 시작할 때 가장 먼저 읽어야 할 문서들입니다.

## 어떤 문서가 들어갈까요?

예를 들어:
- 시스템 전체 개요
- 빠른 시작 가이드
- 사용 방법 설명
- 핵심 개념 소개

## LLM 사용 지침

- **세션 시작 시**: 가장 먼저 이 폴더의 문서를 읽어 전체 맥락 파악
- **방향 설정**: 사용자 요청을 받으면 여기서 어떻게 접근할지 확인
- **라우팅 시작점**: 구체적인 문서는 routing 폴더를 통해 찾아감
