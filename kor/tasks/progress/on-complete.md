# Task 완료 시

> Task 완료 시 수행해야 할 절차

## 1. 완료 체크리스트

- [ ] Task 목표 달성 확인
- [ ] 관련 문서 업데이트 필요 여부 확인 (routing/sync-rules.md 참조)
- [ ] 테스트/검증 수행
- [ ] 부작용 없는지 확인

## 2. current-state.md 업데이트

```markdown
## 완료된 작업
- [x] Task 제목
  - 완료 일시: YYYY-MM-DD HH:MM
  - 주요 변경사항: ...
  - 관련 파일: ...
```

## 3. 사람에게 요약 제공

```
완료한 작업:
- 무엇을 했는지
- 어떤 파일이 변경되었는지
- 확인해야 할 사항 (있다면)
```
