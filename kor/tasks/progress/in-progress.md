# Task 진행 중

> Task를 수행하는 동안 따라야 할 규칙

## 1. 단계별 진행

```
1. 각 단계를 명확히 구분
2. 한 단계씩 완료 확인
3. 중간 결과를 사용자에게 보고
```

## 2. 문서 참조

```
1. routing/main-routing-map.md에서 필요한 문서 확인
2. workflows/에서 해당 작업의 프로세스 확인
3. memory/에서 구체적 지식 참조
```

## 3. 진행 상황 기록

```
- 중요한 결정이나 변경사항 메모
- 다음 단계 명시
- 막힌 부분이나 의문점 기록
```

## 체크리스트

- [ ] 단계별로 진행
- [ ] 필요한 문서 참조
- [ ] 중요 결정 기록
