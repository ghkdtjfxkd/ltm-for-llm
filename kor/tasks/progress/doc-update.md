# 문서 업데이트

> Task 수행 중 문서를 업데이트하는 규칙

## 언제 업데이트하는가?

1. **Task 시작 전**: current-state.md에 "진행 중" 표시
2. **중요 단계 완료 시**: 진행 상황 업데이트
3. **Task 완료 시**: "완료됨" 표시 + 요약
4. **Task 실패 시**: "실패" 표시 + 이유

## 관련 문서 동기화

```
1. routing/sync-rules.md 확인
2. 수정한 문서와 연결된 문서 목록 파악
3. 필요한 문서 업데이트
4. 라우팅 맵 갱신 필요 여부 확인
```

## 업데이트 원칙

- 변경사항은 구체적으로 기록
- 이유를 함께 적기
- 관련 파일 경로 명시
- 날짜/시간 포함
