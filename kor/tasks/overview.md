# Tasks - 작업 관리

## 이 폴더의 역할

프로젝트의 작업(task)을 관리하고 추적합니다.

사람이 진행 상황을 파악하고, AI가 작업을 체계적으로 수행하도록 돕습니다.

## 포함된 문서

### `current-state.md` (사람용)
- 현재 진행 중인 작업들
- 완료된 작업들
- 다음에 할 일
- 세션을 다시 시작할 때 여기부터 읽기

### `ai-task-protocol.md` (AI용)
- Task를 어떻게 처리할지 규칙
- 시작, 진행, 완료, 실패 시 절차
- Task 분리 및 롤백 기준

## 워크플로우

1. **사람**: 새 작업 요청
2. **AI**: `current-state.md` 읽고 현재 상황 파악
3. **AI**: `ai-task-protocol.md` 규칙에 따라 작업 수행
4. **AI**: 완료 시 `current-state.md` 업데이트
5. **사람**: 업데이트된 문서 확인, 다음 작업 지시

## 핵심 원칙

- **작은 단위**: Task는 명확하고 작게
- **문서화**: 진행 상황을 항상 기록
- **롤백 가능**: 실패하면 되돌릴 수 있게
- **연속성**: 세션이 바뀌어도 이어서 할 수 있게
