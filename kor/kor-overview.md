# 한국어 문서

LTM for LLM 프로젝트의 한국어 문서입니다.

## 디렉토리 구조

```
kor/
├── meta/               # 시스템 전체 가이드
├── templates/          # 문서 작성 템플릿
├── routing/            # 문서 찾기 가이드
├── workflows/          # 작업 방식 정의
└── memory/             # 지식 저장소
    ├── architecture/
    ├── conventions/
    ├── patterns/
    ├── troubleshooting/
    └── domain-knowledge/
```

## 각 폴더 역할

- **meta**: 전체 시스템 사용법과 개요
- **templates**: 새 문서 작성할 때 사용할 양식
- **routing**: 상황별로 어떤 문서를 읽을지 안내
- **workflows**: 작업을 어떤 순서로 진행할지 정의
- **memory**: 실제 지식과 정보 저장

## 시작하기

1. `meta/` 폴더의 문서부터 읽기
2. 필요한 상황에 맞게 `routing/` 참조
3. 작업 시작 시 `workflows/` 확인
4. 구체적 정보는 `memory/`에서 찾기
